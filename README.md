# AI Lab Launcher – Windows & Linux

Questo repository contiene due script di avvio per l'ambiente AI Lab, compatibili con Windows e Linux.

## Funzionalità principali

- Attivazione automatica dell'ambiente Conda `ai-lab`
- Accesso diretto alla cartella di lavoro `AI_Lab`
- Avvio di Jupyter Notebook
- Barra di caricamento fluida per ogni fase

Perfetti per un avvio ordinato, visivamente chiaro e senza errori.

## Script inclusi

- `launch_ai_lab.bat` – versione per Windows
- `launch_ai_lab.sh` – versione per Linux

> Assicurati che l'ambiente Conda sia installato e configurato correttamente prima dell'uso.
